🥷 About Ninja Babu

Ninja Babu is a playful, interactive front-end experience inspired by fast-paced slicing games, built with a twist of humor and unexpected behavior. What starts as a simple fruit-slicing challenge gradually evolves into something more unpredictable — reflecting how design can surprise, mislead, and engage users beyond expectations.

This project focuses on combining clean UI design, smooth animations, and creative interaction logic to deliver a fun and memorable user experience.
